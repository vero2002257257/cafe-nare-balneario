🚀 INSTALACIÓN RÁPIDA (3 PASOS):

1️⃣ INSTALAR DEPENDENCIAS:

   npm install

2️⃣ INICIAR APLICACIÓN:
   npm start

3️⃣ ABRIR EN NAVEGADOR:
   http://localhost:3000

¡LISTO! Ya tienes tu sistema funcionando.

🔄 CONFIGURACIÓN AVANZADA (OPCIONAL):

Para usar Google Sheets en lugar de Excel:

1. Copiar env.example a .env
2. Cambiar USE_GOOGLE_SHEETS=true
3. Obtener credenciales de Google Cloud
4. Ejecutar: node scripts/setup-google-auth.js

Ver README.md para instrucciones detalladas.

============================================================

❓ SOLUCIÓN DE PROBLEMAS:

Error "Puerto en uso":
- Windows: netstat -ano | findstr :3000
- Mac/Linux: lsof -ti:3000 | xargs kill -9

No abre la aplicación:
1. Verificar que Node.js esté instalado
2. Ejecutar "npm install" primero
3. Abrir http://localhost:3000
